Setup notes for testing

**IMPORTANT - UPDATE IP Address to RECORDING LOCATION**

- Create VPC
- Create subnet
- Create Internet Gateway
- Create route in Route Table linking IGW to VPC
- Create Security Group - set Inbound Rules to SSH/22 for **my IP**, Outbound rules to all for **my IP**
- Create key pair - download to local and upload to CloudShell

- Create EC2 instance using key pair and named VPC
- Test connectivity 
    - via local SSH (terminal) - chmod on *.pem and cd to file location BEFORE SSH
    - with Cloud Shell 
        - upload key (*.pem) + chmod it, 
        - then get IP EACH SESSION! (curl https://icanhazip.com/v4) and update security group inbound rule for SSH (22) with that IP
